# Keto Plan

Prosta aplikacja PWA z jadlospisem, trackerem wody i przypomnieniami.

## Publikacja na GitHub Pages

1. Utworz nowe publiczne repozytorium na GitHubie, np. `keto-plan`.
2. Wgraj wszystkie pliki z tego folderu do glownego katalogu repozytorium.
3. Wejdz w `Settings` -> `Pages`.
4. W `Build and deployment` wybierz `Deploy from a branch`.
5. Wybierz branch `main` oraz folder `/root`.
6. Zapisz i poczekaj chwile na adres strony.

Adres bedzie wygladal podobnie do:

`https://twoj-login.github.io/keto-plan/`

Na telefonie otworz ten adres i dodaj aplikacje do ekranu glownego.
